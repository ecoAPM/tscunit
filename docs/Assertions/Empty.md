## Assertion: Empty 

Asserts that an array is empty 

### Example 

```ts 
this.assert.empty(array);
``` 

### Conditions 

Passes if `array` contains zero elements

Fails if `array` contains any elements 

### Parameters 

| Name | Description | 
|---|---| 
| `array` | the array to check |
| `message` | (optional) message to display on failure |
